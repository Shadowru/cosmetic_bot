# План сайта V1: Post-Procedure Store

**Дата:** май 2026 | Ниша: домашний уход после косметологических процедур

## Концепция одной фразой

> Навигация по процедуре, не по продукту. Человек приходит с проблемой «я только что сделала лазер», а не «хочу купить крем».

## Страница за страницей

### / — Главная

**Hero-блок:**
```
Заголовок:   «Сделали процедуру? Подберём уход за 60 секунд»
Подзаголовок: «Неправильный крем после лазера — это пигментация.
               Правильный — закрепление результата на месяцы»
CTA:          [Выбрать процедуру ↓]
```

**Быстрый выбор (6 карточек-иконок):**
- После лазера → `/posle-lazera/`
- После биоревитализации → `/posle-biorevit/`
- После пилинга → `/posle-pilinga/`
- После RF-лифтинга → `/posle-rf/`
- После дермароллера → `/posle-dermarollera/`
- После чистки → `/posle-chistki/`

1. Какую процедуру делали?

- Счётчик: «Помогли N клиентам после процедур»

**Топ-наборы** (3 карточки):

- После биоревитализации: набор 7 дней — X ₽
- Универсальный реабилитационный — X ₽

**Блок страха / FAQ:**
- «Почему обычный крем навредит?»
- «Как понять что уход правильный?»
- «Что если началась реакция?»

**Партнёрская плашка внизу:**
«Вы косметолог или клиника? → Подключитесь к партнёрской программе»

### /posle-lazera/ — После лазерной шлифовки

**URL-структура внутри:**
```
/posle-lazera/                    — категория + гид
/posle-lazera/den-1-3/            — первые 3 дня (экстренный уход)
/posle-lazera/den-4-14/           — восстановление
/posle-lazera/posle-zazhivleniya/ — поддержание результата
```

**Контент страницы:**
- H1: «Уход после лазерной шлифовки лица: что делать по дням»
- Блок «По дням»: день 1 / дни 2–3 / дни 4–7 / дни 8–14
- Блок «Что НЕЛЬЗЯ»: ретинол, кислоты, скраб, тепло, спирт
- Блок «Что НУЖНО»: список средств с объяснением компонентов
- Товары с фильтром «для этой процедуры»
- SEO-блок «Вопросы и ответы»
- CTA: «Собрать набор на 14 дней»

**SEO-цели (HIGH запросы):**
- что делать после лазерной шлифовки
- что делать после лазерной шлифовки лица
- что нельзя делать после лазерной шлифовки лица
- крем после лазерной шлифовки
- крем после лазерной шлифовки для лица
- восстановление после лазерного пилинга

**Инсайт — конкурент «Аргосульфан»:**
Яндекс suggest показывает «аргосульфан крем после лазерной шлифовки» в топе.
Это аптечный препарат, не профессиональная косметика.
→ Добавить блок: «Профессиональная альтернатива аргосульфану» — сильный SEO + конверсия.

### /partneram/ — Для клиник

**Оффер для косметолога:**
```
«Ваши клиенты всё равно что-то покупают после процедуры.
 Пусть покупают правильное — и вы получаете 10–15% с каждого заказа.»
```

**Блоки страницы:**
- Как работает: 3 шага (договор → карточки с QR → комиссия)
- Что вы получаете: карточки, QR, UTM-ссылка, личный кабинет со статистикой
- Пример: «50 процедур в месяц × 20% конверсия × 4 500 ₽ = 45 000 ₽ доп. дохода»
- Форма заявки: имя + клиника + телефон

## Технический стек — рекомендации

| Вариант | Когда | Плюсы | Минусы |
|---|---|---|---|
| **Tilda + Ecwid** | MVP / тест | Быстро, дёшево, без разработчика | Ограничения SEO-гибкости |
| **WordPress + WooCommerce** | До 500 SKU | Полный SEO-контроль, плагины | Нужен разработчик |

## SEO-требования (технические)

- ЧПУ по схеме `/posle-[procedura]/`
- H1 = целевой запрос (HIGH из suggest)
- Title = `[Запрос] — PostCare` (до 60 символов)
- Description = польза + CTA (до 160 символов)
- Schema.org: Product + FAQPage + BreadcrumbList
- Скорость: < 2 сек LCP (критично для мобильных)
- Sitemap.xml + robots.txt с первого дня
