# Бизнес-план: магазин уходовой косметики Post-Procedure

**Дата:** май 2026 | Ниша: профессиональный домашний уход после косметологических процедур

---

## Почему ниша работает

### Цифры рынка-донора (клиентская база уже есть)

| Тип процедур | Объём рынка 2025 | Кол-во процедур | Рост |
|---|---|---|---|
| Аппаратная косметология | 244,8 млрд руб. | 23,1 млн | +40% г/г |
| Инъекционная косметология | 59 млрд руб. | 11,5 млн | +15–20% г/г |
| **Итого** | **~304 млрд руб.** | **~34,6 млн** | |

- Биоревитализация — лидер роста: **+46%**, 3,8 млн процедур
- Лазер и RF-лифтинг: **70% всех аппаратных манипуляций**
- Оборот салонов и клиник: **+18% в 2025**

### Механика ниши

Человек потратил на процедуру **5 000–50 000 ₽**. Выходя из клиники, он:

1. Получает от косметолога список ограничений («не мочить», «не использовать кислоты»)
2. Получает бумажку с несколькими рекомендованными средствами — но **где их купить, не объясняют**
3. Дома гуглит: «что можно после лазера», «чем мазать после биоревитализации»
4. Попадает на форумы и статьи клиник — и **не находит магазина**, который продаёт именно то, что нужно

Это идеальная точка входа: клиент горячий, мотивированный не испортить результат процедуры за 30 000 ₽, покупает быстро и берёт сразу несколько средств.

## Концепция

**Название рабочее:** PostCare / AfterGlow / ВОС.КО / Реабилитация.Рф — под выбор

**Позиционирование:**
> «Магазин домашнего ухода после косметологических процедур. Берёте — по процедуре, не наугад.»

**Главный принцип навигации:** не «кремы / сыворотки / маски», а **«после лазера / после пилинга / после инъекций / после RF / после дермароллера»**

**Целевая аудитория:**
- Женщины 28–55 лет, делающие косметологические процедуры 2–6 раз в год
- Средний чек процедуры: 8 000–30 000 ₽
- Готовность платить за «правильный» уход: 3 000–8 000 ₽ за набор
- Живут в городах 300 000+, активно ищут информацию онлайн

## Продуктовая линейка (что продавать)

### Что нужно коже после каждой процедуры

| Процедура | Реабилитация | Что нужно |
|---|---|---|
| Лазерная шлифовка / пилинг | 7–21 день | Барьерный крем + регенерант + SPF50+ + мягкий очиститель |
| Биоревитализация / мезотерапия | 3–7 дней | Антисептик + заживляющий крем + увлажнение без кислот |
| RF-лифтинг / HIFU | 5–14 дней | Успокаивающие сыворотки + питание + SPF |
| Ботокс / филлеры | 1–3 дня | Мягкое очищение + увлажнение (без массажа, без тепла) |
| Химический пилинг | 7–14 дней | Регенерант + барьерное масло + SPF50+ |
| Дермароллер / микронидлинг | 3–7 дней | Успокаивающая сыворотка + барьерный крем (без кислот) |
| Фракционный лазер | 14–28 дней | Полный протокол: 5–6 SKU |

### Ключевые категории SKU

**Обязательные (базовый ассортимент, 30–50 SKU):**
- Регенерирующие / барьерные кремы (с церамидами, пантенолом, мадекассозидом)
- Мягкие очищающие пенки (без SLS, без отдушек, рН 5.5)
- SPF50+ для чувствительной / восстанавливающейся кожи
- Успокаивающие сыворотки (центелла, ниацинамид, гиалуроновая кислота)
- Заживляющие кремы-мази (bepanthen-аналоги + проф. варианты)
- Увлажняющие маски (без кислот, без спирта)

**Дополнительные (расширенный ассортимент):**
- Антисептики и тоники для свежезаживающей кожи
- Peptide-сыворотки для поддержания результата процедуры
- Средства для профилактики пигментации (витамин С — но только после заживления)
- Охлаждающие гели (алоэ, огурец) — первые дни после процедуры

**Готовые наборы (главный конверсионный продукт):**
- «После лазера: набор на 14 дней» — 4 SKU + инструкция по дням
- «После биоревитализации: первые 7 дней» — 3 SKU
- «Базовый реабилитационный набор» — универсальный, 3 SKU
- «Полный протокол после фракционного лазера» — 5–6 SKU

## SEO-стратегия

### Логика: человек ищет сразу после процедуры — горячий трафик

Он гуглит не «купить крем», а «что делать после лазера» — информационный запрос с транзакционным намерением.

### Кластеры запросов (проверить в Wordstat)

**Лазер и шлифовка:**
```
что делать после лазерной шлифовки
уход после лазера лица
чем мазать после лазера
что нельзя после лазерного пилинга
крем после лазерной шлифовки
реабилитация после лазера дома
```

**Биоревитализация:**
```
уход после биоревитализации
что нельзя после биоревитализации
крем после биоревитализации
что можно наносить после биоревитализации
```

**Химический пилинг:**
```
уход после химического пилинга
что после пилинга нельзя
крем после пилинга
чем увлажнять после пилинга
```

**Дермароллер:**
```
уход после дермароллера
что наносить после дермароллера
можно ли крем после дермароллера
```

**RF и аппаратная косметология:**
```
уход после RF-лифтинга
уход после HIFU
что после аппаратной косметологии
```

### Формула SEO-статьи (одна процедура = одна статья)

1. **Что происходит с кожей** после этой процедуры (физиология)
2. **По дням**: что делать день 1 / дни 2–3 / дни 4–7 / дни 8–14
3. **Что НЕЛЬЗЯ**: ретинол, кислоты, скрабы, спирт, тепло — с объяснением почему
4. **Что НУЖНО**: список средств с объяснением компонентов
5. **Товары**: конкретные SKU из нашего магазина с кнопкой «В набор»
6. **Ошибки**: топ-5 ошибок в уходе после [процедуры]
7. **FAQ**: «Почему кожа шелушится?», «Когда вернуться к обычному уходу?»
8. **CTA**: «Собрать набор под вашу процедуру за 3 минуты»

**Контент-план минимум:** 12 статей (по одной на каждую процедуру) + 5–7 по компонентам (церамиды, пантенол, SPF и т.д.)

## Партнёрский канал (B2B2C — ключевой)

Это главное конкурентное преимущество бизнес-модели.

### Механика

Косметологические клиники → рекомендуют клиентам ваш магазин вместо «поищите в интернете»

**Что предлагаем клинике:**
- Карточки-инструкции с QR-кодом «куда идти за уходом» — клиника раздаёт пациентам
- Партнёрская ссылка с UTM → клиника видит, сколько клиентов перешло и купило
- Комиссия 10–15% с каждого заказа клиента клиники
- Возможность для клиники: брендированные наборы под названием клиники

**Что это даёт вам:**
- Горячий трафик с нулевой стоимостью привлечения
- Клиент приходит уже с доверием (косметолог рекомендовал)
- Конверсия таких клиентов: ожидаемо 30–50% (против 2–3% у холодного трафика)
- LTV: клиенты клиники делают процедуры 2–6 раз в год → каждый раз покупают набор

**Как начать:**
1. Выбрать 5–10 клиник в своём городе
2. Встретиться лично с главным косметологом
3. Предложить: «Ваши пациенты всё равно покупают что-то где-то — пусть покупают правильное у нас, и вы получаете комиссию»
4. Дать пробные наборы на тест

**Масштаб:** в Москве ~3 000+ косметологических клиник и кабинетов. Даже 50 партнёрских клиник по 200 процедур/мес = **10 000 горячих потенциальных клиентов в месяц**.

## Воронка продаж и LTV

### Первая покупка (горячий лид)

```
Процедура в клинике
  ↓ Косметолог даёт карточку с QR / человек гуглит сам
  ↓ Попадает на статью «Уход после [процедуры]»
  ↓ Читает по дням → видит товары
  ↓ Собирает набор за 3 минуты
  ↓ Покупает (AOV: 3 500–7 000 ₽)
```

### Удержание (LTV)

- **День 3 после заказа** (WhatsApp/Telegram): «Как кожа? Нет реакций? Если что — мы здесь»
- **День 10**: «Через 4–7 дней можно вернуться к обычному уходу. Вот что добавить следующим шагом»
- **Через 6–8 недель**: «Скоро следующая процедура? Набор уже можно собрать заранее»
- **Триггер повтора**: большинство процедур повторяются каждые 1–6 месяцев — напоминание = повторная покупка

### Финансовые показатели (ориентир)

| Метрика | Цель |
|---|---|
| AOV (средний чек) | 4 500 ₽ |
| Доля наборов в выручке | > 50% |
| Конверсия сайта (SEO-трафик) | 3–5% |
| Конверсия партнёрского трафика (клиники) | 25–40% |
| Повторная покупка (6 мес) | > 35% |
| LTV 12 мес | > 10 000 ₽ |
| CAC (стоимость клиента) | < 800 ₽ |

## Roadmap: первые 90 дней

### Месяц 1 — Фундамент

**Неделя 1–2: исследование и продукт**
- [ ] Обойти 10–15 косметологических клиник своего города, поговорить с косметологами
- [ ] Выяснить: что они сейчас рекомендуют пациентам? Что те покупают не то?
- [ ] Составить список 30 SKU под 6 основных процедур
- [ ] Найти поставщиков (Inspira, Beauty Style, Ангиофарм, проф. дистрибьюторы)

**Неделя 3–4: сайт и контент**
- [ ] Запустить сайт с навигацией по процедурам (не по категориям)
- [ ] Написать 6 базовых SEO-статей (лазер, биоревитализация, пилинг, RF, дермароллер, ботокс)
- [ ] Снять 10 видео применения к ключевым SKU
- [ ] Подготовить карточки с QR для партнёрских клиник

### Месяц 2 — Партнёрства и первые продажи

- [ ] Подписать 5–10 партнёрских клиник (договор + карточки + UTM-ссылки)
- [ ] Запустить Яндекс.Директ по запросам «уход после [процедура]»
- [ ] Написать ещё 6 SEO-статей (ошибки, компоненты, FAQ)
- [ ] Настроить WhatsApp-сопровождение первых 50 клиентов (вручную)
- [ ] Собрать 10–20 отзывов с фото (скидка 15% за фото через 2 недели)

### Месяц 3 — Масштаб

- [ ] Довести до 20–30 партнёрских клиник
- [ ] Запустить Telegram-канал: «Уход после процедур — что работает»
- [ ] A/Б тест наборов vs отдельных товаров (гипотеза: наборы конвертят лучше)
- [ ] Замерить LTV первых клиентов — есть ли повторные покупки через 6–8 недель?
- [ ] Разработать брендированные наборы для 2–3 клиник-партнёров (их логотип на упаковке)

## Риски и как снять

| Риск | Вероятность | Как снизить |
|---|---|---|
| Клиники не хотят рекомендовать | Средняя | Предложить комиссию + готовые материалы (не надо думать) |
| Покупатели не ищут в Google | Низкая | Рынок инъекций 34,6 млн процедур/год — ищут точно |
| Конкурент скопирует модель | Средняя (через 12–18 мес) | Быстро занять позиции в SEO + выстроить партнёрскую сеть |
| РРЦ на бренды | Низкая | Post-procedure — это аптечные + проф. бренды, РРЦ мягче |
| Маленький повторный спрос | Низкая | Процедуры повторяются, триггерные сообщения возвращают |

## Главная идея одной фразой

Косметологический рынок России — **34,6 млн процедур в год**. После каждой процедуры пациент выходит из клиники с бумажкой «что нельзя» и без ответа «где купить то, что можно». Это и есть ниша.

Специализированного магазина с навигацией по процедурам не существует. Первый, кто займёт место, получит:
- горячий SEO-трафик без конкуренции
- партнёрский канал через клиники с конверсией 30–40%
- высокий LTV (процедуры повторяются 2–6 раз в год)
