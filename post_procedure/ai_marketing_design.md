# AI-маркетинг: дизайн системы

**Статус:** проект, не реализован  
**Контекст:** Post-Procedure Store, ниша постпроцедурного ухода

---

## Что система делает

Генерирует маркетинговый контент трёх типов на основе данных из `products.json` и `keyword_report.xlsx`, отправляет на проверку в Telegram одним нажатием, публикует после одобрения.

```
Данные (products.json, keywords)
        ↓
   Генератор контента (Claude API)
        ↓
   Очередь на проверку
        ↓
   Превью в Telegram → [✅ Опубликовать] [❌ Отклонить] [✏️ Переделать]
        ↓
   Публикация (Telegram канал / YouTube Shorts)
```

Ключевой принцип: **AI генерирует, человек одобряет одним тапом.** Полный автопилот без проверки — риск.

---

## Три типа контента

### 1. Telegram-посты (текст)

**Источник данных:** `products.json`, `keyword_report.xlsx`, темы из `website_plan.md`  
**Модель:** Claude API (claude-sonnet)  
**Промпт-система:** шаблоны по типу поста (образовательный / миф / практика / вопрос-ответ) + стиль из `tg_posts_week1.md`  
**Объём:** 1 пост в день  
**Стоимость:** ~$0.02 за пост

Что передаётся в промпт:
- Тип поста (ротация по расписанию)
- Процедура дня (ротация по 6 процедурам)
- Релевантные HIGH-запросы из keyword_report
- Примеры постов из `tg_posts_week1.md` как few-shot

---

### 2. Shorts / Reels (видео, 30–60 сек)

**Конвейер:**

```
Claude API → скрипт (хук + 3 блока + CTA)
     ↓
Yandex SpeechKit / ElevenLabs → войсовер MP3
     ↓
Pexels API → стоковые клипы по ключевым словам (skincare, face, cream)
     ↓
FFmpeg → сборка: клипы + войсовер + субтитры + текст-оверлей + логотип
     ↓
MP4 файл → очередь на проверку
```

**Форматы роликов (шаблоны):**

| Шаблон | Пример | Хук |
|---|---|---|
| По дням | «Что делать в первые 3 дня после лазера» | «Ты только что с лазера. Вот что будет если...» |
| Ошибки | «3 ошибки после биоревита» | «Синяки не проходят? Вот почему» |
| Нельзя | «Что нельзя после пилинга» | «Косметолог не предупредила — я сделала это» |
| Продукт | «Чем отличается Bioderma от аптечного крема» | «Ты платишь за бренд или за состав?» |

**Стек:**
- Скрипт: Claude API (200–300 слов = 60 сек)
- Голос: Yandex SpeechKit (русский, женский) — дешевле ElevenLabs для RU
- Стоковое видео: Pexels API (бесплатно)
- Субтитры: Whisper (транскрибирует войсовер → SRT)
- Монтаж: FFmpeg + MoviePy
- Текстовые оверлеи: Pillow

**Объём:** 3–4 ролика в неделю  
**Стоимость:** ~$0.30–0.50 за ролик (основное — войсовер)

---

### 3. SEO-статьи (текст, 1500–3000 слов)

**Источник:** HIGH-запросы из `keyword_report.xlsx` + формула из `website_plan.md`  
**Модель:** Claude API (claude-opus для длинных статей)  
**Формула:** проблема → физиология → по дням → что нельзя → что нужно → продукты → ошибки → FAQ  
**Объём:** 1–2 статьи в неделю  
**Стоимость:** ~$0.10–0.20 за статью

## Механизм проверки (Review Bot)

Существующий `@nenaugad_bot` расширяется режимом для владельца.

**Флоу:**

```
Генератор → бот отправляет превью владельцу в личку:

┌─────────────────────────────┐
│ 📝 ПОСТ — Понедельник        │
│ Процедура: биоревитализация  │
│                              │
│ [текст поста целиком]        │
│                              │
│ [✅ Опубликовать] [❌ Пропустить] [🔄 Переделать]
└─────────────────────────────┘
```

Для видео — бот присылает MP4 превью + скрипт, те же кнопки.

**Команды владельца:**
- `/queue` — посмотреть что в очереди
- `/generate post laser` — сгенерировать пост по процедуре вне расписания
- `/generate video piling` — сгенерировать ролик

SEO-статьи генерируются в фоне, в очередь для ручной публикации на сайт.

## Технический стек

| Компонент | Инструмент | Стоимость |
|---|---|---|
| Генерация текста | Claude API (Sonnet/Opus) | ~$0.01–0.20 / запрос |
| Голос для видео | Yandex SpeechKit | ~0.003 $/сек |
| Стоковые клипы | Pexels API | Бесплатно |
| Транскрипция → субтитры | Whisper (local) | Бесплатно |
| Монтаж видео | FFmpeg + MoviePy | Бесплатно |
| Текст-оверлеи | Pillow | Бесплатно |
| Хранение очереди | SQLite или JSON-файлы | Бесплатно |
| Публикация | Telegram Bot API + YouTube Data API | Бесплатно |

**Итого на месяц при полном расписании:**
- 30 постов × $0.02 = $0.60
- 14 роликов × $0.40 = $5.60
- 6 статей × $0.15 = $0.90
- **Итого: ~$7–10/мес**

## Что НЕ автоматизируем

- Публикация без проверки человека
- Генерация рекомендаций конкретных продуктов (только общие советы — ответственность)
- Видео с лицами/людьми (юридические риски дипфейков)
- Медицинские утверждения
